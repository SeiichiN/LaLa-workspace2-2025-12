モンスターの設計

Monsterクラス
  抽象クラス

フィールド
  name   :String
  suffix :char
  hp     :int

コンストラクタ
  引数: String
        char

メソッド
  attack(Player) -- デフォルトメソッドとしておく。


Goblinクラス・Slimeクラス
  具象クラス
  Monsterクラスを継承

コンストラクタ
  引数: String
        char

メソッド
  attack(Player) -- これはオーバーライドしなくてもよい。
